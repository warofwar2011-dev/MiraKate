# MiraKate
Messagerie sécurisée P2P avec échange de clés local et protocole simple (PING/PONG/MSG/ACK)

Contributions bienvenues
MiraKat est ouvert à tous :
  développeurs (débutants à confirmés),
  passionnés de sécurité,
  gens qui aiment la crypto post-quantique,
  designers UI/UX,
  rédacteurs de docs.
Idées de contributions :
  Proposer un premier prototype en Python / Go / Rust.
  Expérimenter avec liboqs
  pour la partie PQC.
  Implémenter le protocole PING/PONG/MSG/ACK.
  Gérer le stockage local des messages.
  Ajouter un client minimal (CLI, puis UI).
